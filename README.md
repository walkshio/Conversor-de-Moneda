# 💰 Conversor de Monedas - Alura Challenge

¡Bienvenido/a! Este proyecto es una aplicación de consola en Java que permite realizar conversiones de divisas en tiempo real. Fue desarrollado como parte del desafío de programación de **Alura Latam** (Oracle Next Education).

## 🚀 Funcionalidades
- **Conversiones disponibles:** - Dólar (USD) ↔ Sol Peruano (PEN)
  - Dólar (USD) ↔ Peso Argentino (ARS)
  - Dólar (USD) ↔ Real Brasileño (BRL)
  - Dólar (USD) ↔ Peso Colombiano (COP)
- **Datos en tiempo real:** Uso de **ExchangeRate-API** para obtener las tasas de cambio más recientes.
- **Interfaz amigable:** Menú interactivo por consola con validación de opciones.
- **Precisión Financiera:** Los resultados se muestran formateados a **2 decimales**.

## 🛠️ Tecnologías Utilizadas
- **Java 21**: Uso de `HttpClient`, `HttpRequest` y manejo de excepciones.
- **Maven**: Gestión profesional de dependencias y estructura de proyecto estándar.
- **Gson 2.10.1**: Biblioteca de Google para el parseo de datos JSON provenientes de la API.
- **Git/GitHub**: Control de versiones y despliegue del código.

## 📁 Estructura del Proyecto
El proyecto sigue el estándar de Maven:
- `src/main/java/`: Contiene la lógica principal (`Principal.java`).
- `pom.xml`: Archivo de configuración con las dependencias del proyecto.
- `.gitignore`: Configuración para excluir archivos temporales de IDEs.
